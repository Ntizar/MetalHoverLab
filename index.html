MetalHoverLab

Relieves 3D que el cursor metaliza

Sube una escena blanca o marmol tipo bajorrelieve. El laboratorio reconstruye un relieve util a partir de sus sombras para que el puntero anada color metalico, brillo especular y profundidad aparente.

Esta version ya no pinta una silueta plana. Mantiene la imagen original y calcula un relight localizado sobre sus volumenes para acercarse al efecto de figuras con altura que se encienden al pasar por encima.

Funciona mejor con escenas claras, fondo liso y volumenes blancos o gris muy suave. Se puede probar con JPG, PNG o WEBP.
