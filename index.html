Relieves 3D que el cursor metaliza

Sube una escena blanca o mármol tipo bajorrelieve. El laboratorio reconstruye un relieve útil a partir de sus sombras para que el puntero añada color metálico, brillo especular y profundidad aparente.

Esta versión ya no pinta una silueta plana. Mantiene la imagen original y calcula un relight localizado sobre sus volúmenes para acercarse al efecto de figuras con altura que se encienden al pasar por encima.

Funciona mejor con escenas claras, fondo liso y volúmenes blancos o gris muy suave. Se puede probar con JPG, PNG o WEBP.

Tutorial rápido

Cómo llevarlo a una web o proyecto

Puedes usarlo en HTML estático, CMS que permitan scripts inline o proyectos React/Next/Vue extrayendo el runtime a un archivo JS propio.

01

Prepara la escena

Usa una imagen clara de relieve. Cuanto más visibles sean las sombras del volumen, mejor responderá la luz.

02

Exporta y copia

Pulsa Actualizar export y luego copia el snippet o el HTML completo según el flujo de tu proyecto.

03

Pega y escala

Escala el efecto editando --mhl-max-width, --mhl-padding y --mhl-radius en la raíz exportada.

04

Integra en frameworks

En React o Next, úsalo dentro de un componente cliente o mueve el runtime exportado a un módulo JS del proyecto.
