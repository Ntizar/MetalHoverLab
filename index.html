MetalHoverLab

Hover metalizado exportable para cualquier web

Sube un fondo y una figura blanca en PNG o SVG con transparencia. Ajusta el efecto y exporta un snippet HTML/CSS/JS portable para integraciones donde se permitan estilos y script inline.

La referencia inspeccionada parece usar Rive sobre canvas. Este laboratorio la traduce a una version portable con mask-image, gradientes metalicos y brillo reactivo al puntero.

Si no subes nada, se usa un gradiente editorial de prueba. Si la imagen es muy grande, el export final pesara bastante.

Ideal: PNG o SVG con transparencia. El efecto usa la silueta como mascara y embebe el asset en el snippet exportado.
