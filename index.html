Tutorial

Puedes usar este efecto en HTML estático, CMS que permitan scripts inline o proyectos React/Next moviendo el runtime a un archivo propio.

01

Prepara la escena

Usa una imagen clara de relieve. Cuanto más visibles sean las sombras del volumen, mejor responderá la luz.

02

Exporta y copia

Pulsa Actualizar export y luego copia el snippet o el HTML completo según tu flujo.

03

Pega y escala

Escala el efecto editando --mhl-max-width, --mhl-padding y --mhl-radius.

04

Integra en frameworks

En React o Next, usa el export dedicado o mueve el runtime exportado a un helper.
